| -------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------ |

| [Essential Linux Device Drivers](https://www.amazon.com/Essential-Device-Drivers-Sreekrishnan-Venkateswaran/dp/0132396556) | Introduction to driver development for major subsystems         |
